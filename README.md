# HearDiseasePredictionUsingFFN
❤️ Heart Disease Prediction using Feed Forward Neural Network (FFN)
📌 Project Overview

This project focuses on predicting the presence of heart disease using a Feed Forward Neural Network (FFN).
The model is trained on a structured medical dataset and learns patterns from patient health attributes to make accurate predictions.

The entire implementation is done using Python and PyTorch, and the workflow is demonstrated in a Jupyter Notebook.

🎯 Objectives

To understand and implement a Feed Forward Neural Network from scratch

To preprocess and analyze a medical dataset

To train and evaluate a neural network for binary classification

To visualize data insights before model training

🧠 Model Used

Feed Forward Neural Network (FFN)

Fully connected layers

ReLU activation function

Sigmoid activation at the output layer for binary classification

Binary Cross-Entropy loss function

Adam optimizer

📊 Dataset Description

The dataset contains medical attributes commonly used to diagnose heart disease, such as:

Age

Sex

Chest pain type

Resting blood pressure

Cholesterol level

Fasting blood sugar

ECG results

Maximum heart rate

Exercise-induced angina

The target variable indicates whether a person has heart disease (1) or no heart disease (0).
